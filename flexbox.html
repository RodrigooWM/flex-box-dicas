Flex-Box / Git básico:

display: flex;
Identificar que a tag vai ser o elemento container do flexbox

flex-direction: row | row-reverse | column | column-reverse | inherit | initial | unset ;
Diz a direção do eixo principal

justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly | initial | inherit;
Alinha os elementos no eixo principal

align-items: stretch | center | start | end;
Alinhar os elementos no eixo x.

flex-wrap: nowrap | wrap;
Quebra (ou não) a linha se não houver espaço dentro do container.

flex-flow: row wrap | row-reverse nowrap | column wrap-reverse | wrap ;
Definem o flex-direction e o flex-wrap ao mesmo tempo.

align-content: stretch | center | flex-start | flex-end | space-between | space-around | space-evenly | initial | inherit;
Quando há o flex-wrap mas queremos alinhas tudo no eixo x.

gap, row-gap e column-gap: valor numérico;
Fixa o espaço entre os elementos, lado a lado ou entre eles em cada uma das linhas.

position: static | relative | absolute | sticky;
Determina o tipo de posição do elemento. A posição exata é definida através dos atributos "top, bottom, right e left". Também podemos usar o margin e padding a nosso favor, para tornar o visual mais agradável.

Create a new repository on the command line
git init
git add README.md (nome do arquivo e sua extensão)
git commit -m "first commit" (comentário que será atribuído ao arquivo)
git branch -M main
git remote add origin https://github.com/RodrigooWM/nome do repositório.git
git push -u origin main
